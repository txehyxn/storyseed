# StorySeed 사용자 흐름

## 1. 문서 목적

이 문서는 StorySeed에서 사용자가 이야기를 시작하고, 선택을 통해 이야기를 발전시키며, 완성된 결과를 확인하는 전체 흐름을 정의한다.

StorySeed의 핵심 경험은 다음과 같다.

> 사용자가 이야기의 방향을 직접 선택하고,  
> AI는 사용자의 선택을 바탕으로 이야기를 이어간다.

사용자는 단순히 AI가 만든 이야기를 읽는 사람이 아니라, 이야기의 방향을 결정하는 작가가 된다.

---

## 2. 전체 사용자 흐름

```text
메인 화면
    ↓
이야기 시작 방식 선택
    ↓
이야기 기본 설정
    ↓
주인공 설정
    ↓
AI가 이야기 도입부 생성
    ↓
사용자가 선택지 선택 또는 직접 행동 입력
    ↓
AI가 선택 결과를 반영하여 이야기 생성
    ↓
선택과 이야기 생성 반복
    ↓
이야기 종료
    ↓
완성된 이야기 확인
    ↓
선택 리포트 확인
    ↓
이야기 저장
